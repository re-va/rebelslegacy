# Rebel’s Legacy Publishing
### An author-led, independent publishing imprint.

This repository contains the source code for the official Rebel’s Legacy website—a minimalist, three-page portal dedicated to intentional writing and independent publishing.

## 1. Project Intent
This site serves as a digital home for *The Book* and the philosophy of the Rebel's Legacy imprint. It is designed to be high-performance, responsive, and aesthetically restrained.

## 2. Technical Architecture
* **Structure:** Single-file SPA (Single Page Application) architecture.
* **Typography:** EB Garamond (Google Fonts).
* **Styling:** Tailwind CSS (Utility) + Custom CSS (Thematic).
* **Data Capture:** "Hidden" AJAX-based contact form via Formspree API.
* **Hosting:** Optimized for GitHub Pages.

### Data Privacy
The "Notify Me" form collects only essential data (Name, Email, Genre Preference). 
* **Use:** Data is used exclusively for book release updates.
* **Retention:** Information is kept until the notification is sent or a deletion request is made.
* **Third Parties:** No data is sold or shared. Form processing is handled by Formspree under their standard privacy terms.

### Ethics of Intent
In alignment with the imprint’s manifesto, this site uses no tracking cookies, no invasive analytics, and no automated marketing sequences.
